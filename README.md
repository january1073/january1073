- **Auto-complete:** `[TAB]`
- **Cursor**
    - `[CTRL] + A/E` - Start/end of line
    - `[CTRL] + [←]/[→]` - Jump to word start/previous
    - `[ALT] + B/F` - Jump backward/forward word
- **Text edit**
    - `[CTRL] + U/K` - Erase to start/end of line
    - `[CTRL] + W` - Erase word
    - `[CTRL] + Y` - Paste
- **Process**
    - `[CTRL] + C` - Kill
    - `[CTRL] + D` - EOF
    - `[CTRL] + Z` - Suspend
- **Terminal**
    - `[CTRL] + L` - Clear
    - `[CTRL] + [+]/[-]` - Zoom in/out
- **History**
    - `[CTRL] + R` - Search
    - `[↑]/[↓]` - Previous/next
- **App switch:** `[ALT] + [TAB]`
